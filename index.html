GA4 Ecommerce Events

Every Action.
Fully Tracked.

Six standard GA4 ecommerce events fire automatically as users interact with the store.

view_item_list

Product List Viewed

Fires when the homepage loads with all 15 products visible.

add_to_cart

Item Added to Cart

Fires with product name, price, category and quantity selected.

remove_from_cart

Item Removed

Fires when a user removes an item from the cart sidebar.

begin_checkout

Checkout Started

Fires on checkout page load with full cart value and items.

purchase

Order Placed

Fires with unique order ID, revenue, tax, and item breakdown.

remove_from_cart

Cart Cleared

Tracks individual item removals for funnel drop-off analysis.
